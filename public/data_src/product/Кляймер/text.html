Кляммер для крепления керамогранита

Кляммер для крепления керамогранита - элемент, входящий в систему вентилируемых фасадов служит для крепления плиты из керамогранита. Данная продукция выполняется из оцинкованной и нержавеющей стали (AISI 430;304) толщиной 1-1,2 мм.

Выбор материала для изготовления подсистемы в целом зависит зависит от условий эксплуатации, особенностей внешней окружающей среды и климатических условий.

Кляммер для крепления керамогранита монтируется на несущие конструкции вентилируемого фасада при помощи винтов саморезов по металлу, а также вытяжными заклепками.

  • Крепёжный кляммер рядовой
  • Крепёжный кляммер стартовый
  • Крепёжный кляммер концевой
